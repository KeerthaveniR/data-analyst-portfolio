Customer Shopping Behaviour Analysis Dashboard (Power BI)

Overview
Analyzed 99K+ customer transactions using Power BI and Power Query to identify purchasing trends across gender, age group, product category, and payment methods.

Dataset
- 99K+ customer transactions

What I Did
- Cleaned and transformed transaction data using Power Query
- Developed 1 interactive dashboard with 4 DAX-based KPIs (Total Revenue, Total Transactions, Average Order Value, and Total Units Sold) to monitor business performance and customer behavior
- Built 10 analytical visualizations covering purchasing trends across gender, age group, product category, and payment methods

Key Metrics
- ₹251.51M in revenue analyzed
- 299K units sold
- DAX-based KPIs
- 10 visualizations

Files in this folder
- `customer_data_analysis_dashboard` — Power BI file
- `customer_data_analysis_ppt` — project presentation
- `customer_data_analysis_report` — analysis report
- `dataset` — dataset used for the analysis
- `dashboard-screenshots/` — dashboard views and visualizations

Tools
Power BI, DAX, Power Query
